Sobre o curso de ciência de dados

O curso de Ciências de Dados da UEPA é projetado para fornecer aos alunos uma base sólida em matemática, estatística, programação e análise de dados. Ao longo do programa, os estudantes adquirem as habilidades necessárias para coletar, organizar, analisar e interpretar grandes volumes de dados, a fim de extrair informações valiosas, tomar decisões informadas e resolver problemas analíticos complexos. Neste curso, os alunos irão desenvolver uma visão geral sobre a área e como ela se aplica no mercado de trabalho, ultilizando de ferramentas matemáticas e computacionais para a visualização e análise de dados, com trabalhos e projetos direcionados para o desenvolvimento de experiência de cada aluno.
